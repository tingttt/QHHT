# QHHT Guidance Website

Live Website  
https://tingttt.github.io/QHHT/

## Overview

This website is a personal informational site for a Quantum Healing Hypnosis Technique (QHHT) practitioner.  
The goal of the site is to provide a calm and welcoming place where visitors can:

- Learn about QHHT
- Explore helpful resources
- Read answers to common questions
- Contact the practitioner for consultation or session inquiries

The site is designed as a simple, clean "digital introduction" for people who are curious about hypnosis, self-discovery, and inner exploration.

Sessions are currently conducted **in person only in Jersey City, New Jersey**.

---

## Features

### Home Page
Introduces the practitioner, explains the purpose of the website, and provides a brief overview of QHHT.

### About & Experience
Shares the practitioner's personal experience with QHHT, including participation in regression sessions and training that inspired the practitioner to pursue this work.

### Resources
Provides curated materials and links for people who want to learn more about QHHT.

### Q&A
A searchable section containing common questions and helpful answers for people who are new to QHHT.

### Contact
Visitors can reach out to ask questions, request more information, or reserve a session.  
Consultations are free, although responses may sometimes take time due to client volume.

### Location
Sessions are conducted **in person **, and a Google Map is included to help visitors understand the location.

## Technology Stack

This site is intentionally lightweight and simple.

Built using:

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages (hosting)

No frameworks are used so that the site remains fast, easy to maintain, and accessible.
